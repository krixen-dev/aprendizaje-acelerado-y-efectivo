Método Map

En JavaScript, el método map() se usa en arreglos para aplicar una función a cada elemento y crear un nuevo arreglo con los resultados. Importante: este método no cambia el arreglo original, simplemente devuelve uno nuevo con los resultados de aplicar la función a cada elemento.

La sintaxis básica del método map() es la siguiente:
var nuevoArreglo = arregloOriginal.map(function(elementoActual, indice, arregloCompleto) {
// Código para procesar cada elemento del arreglo
// Devolver el resultado de la operación
});

  • elementoActual: El valor del elemento actual que está siendo procesado en el arreglo.
  • índice: El índice del elemento actual en el arreglo.
  • arregloCompleto: El arreglo original que está siendo mapeado.

Ejemplo 1

-multiplicar todos los números de la siguiente lista:
numeros = [1, 2, 3, 4, 5]
