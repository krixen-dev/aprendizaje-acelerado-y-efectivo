for of

for of simplifica la iteración sobre estructuras de datos iterables, como arrays, cadenas, mapas, conjuntos, etc.

La sintaxis básica de for...of es la siguiente:

for (variable of iterable) {
// código a ejecutar en cada iteración
}

Aquí, variable es una variable que tomará el valor de cada elemento en iterable en cada iteración del bucle. iterable es la estructura de datos que se está recorriendo.

Ejemplo 1

let numeros = [1, 2, 3, 4, 5];

for of también se puede utilizar para iterar Strings

mensaje = "hola mundo";

  • for of funciona con cualquier iterable como arrays, mapas, cadenas, etc..
  • No está diseñada para trabajar con objetos normales (plain objects)
