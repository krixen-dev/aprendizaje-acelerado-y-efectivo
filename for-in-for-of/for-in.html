for in

La estructura for...in es un bucle en JavaScript que se utiliza para iterar sobre las propiedades enumerables de un objeto. A diferencia de for...of, que se utiliza para iterar sobre valores de estructuras iterables (como arrays, cadenas, etc.), for...in se utiliza específicamente para objetos y recorre las propiedades del objeto.

sintaxis básica de for...in:

for (variable in objeto) {
// Código a ejecutar en cada iteración
}
  • variable: Una variable que tomará el nombre de cada propiedad durante cada iteración.
  • objeto: El objeto sobre el cual se está iterando.
